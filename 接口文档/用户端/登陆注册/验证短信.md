# 发送短信验证字符串

## 1.接口描述

接口请求域名:user/checkString
发送邮箱验证字符串//每个token{session}具有60s请求时限->找回密码 注册 登录都可使用//手机号判重在这个地方做了
请求方式:get

## 2.输入参数

| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
|  phone  |  是   | String | 电话号码 |

## 3.输出参数

| 参数名称 |  类型  |        描述        |
| :------: | :----: | :----------------: |
| success | int | 状态值,1成功,0失败 |
| info | string | 状态描述 |

## 4.用例

### 输入

```json
{
    "phone":18959082820
}
```

### 输出

```json
{
    "success":1,
    "info":"验证码发送成功"
}
```
